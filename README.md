# Quick Notes App 📝

A simple and modern **Notes Taking Web Application** built using **HTML, CSS, and JavaScript**.
This application allows users to quickly create, edit, and manage notes in a clean and responsive interface.

---

## 🚀 Features

* ➕ Add new notes
* 📝 Edit existing notes
* 🗑 Delete notes
* 🌙 Dark / Light theme toggle
* 💾 Local storage support (notes remain after refresh)
* 📱 Responsive design
* ✨ Smooth UI with modern styling

---

## 🛠 Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive design
* **JavaScript (Vanilla JS)** – App functionality and logic
* **Local Storage API** – Saving notes in the browser

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/KAB145/notes-keeper.git
```

2. Navigate to the project folder

```
cd quick-notes-app
```

3. Open the project

Simply open **index.html** in your browser.

---

## 💡 How It Works

* Users can create notes using the **Add Note button**.
* Notes are displayed in a **responsive grid layout**.
* Each note can be **edited or deleted**.
* Notes are saved using **Local Storage**, so they remain even after refreshing the page.
* The **Theme Toggle button** switches between **Light Mode and Dark Mode**.
